# Ice Cream Linux

Linux chroot instance for Android 4.x devices. (needs root)

Includes Firefox 140, Chrome 112, 'spot' GTK Spotify Client.

- Download and install the Linux Deploy APK.  [Version 2.5.1](https://github.com/meefik/linuxdeploy/releases/tag/2.5.1) is the latest version for Android 4.x.
  
- Download the Ice Cream Linux disk image.

- Download an RDP client to connect to the Linux instance:

  - Android 4.1+ · [Remote Desktop 8 8.1.82.445](https://www.apkmirror.com/apk/microsoft-corporation/microsoft-remote-desktop/microsoft-remote-desktop-8-1-82-445-release/)

  - Android 4.0.3+ · [Remote Desktop 8 8.1.28.2](https://www.apkmirror.com/apk/microsoft-corporation/microsoft-remote-desktop/microsoft-remote-desktop-8-1-28-2-release/)
